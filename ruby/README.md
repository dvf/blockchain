## Installation

1. **require ruby 2.3+**
2. install gems

```
$ cd ruby
$ bundle install
```

3. Run the server:
    * `$ bundle exec ruby app.rb -p 5001` 
    * `$ bundle exec ruby app.rb -p 5002`
